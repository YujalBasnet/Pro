# StreamFlix

StreamFlix is a Netflix-inspired front-end clone built with HTML, CSS, and JavaScript. It is fully responsive and includes a sticky navigation bar, a dynamic hero section, genre filters, horizontal content rails, search, and a mobile-friendly menu.

## Features

- Responsive layout for desktop, tablet, and mobile screens
- Netflix-style hero section with featured title switching
- Search filter for titles, genres, moods, and metadata
- Genre chip filters for fast browsing
- Horizontal content rails with interactive cards
- JavaScript-powered mobile navigation and saved list toggle

## Files

- `index.html` - page structure
- `styles.css` - responsive styling and layout
- `app.js` - interactive behavior and data rendering

## How to Run

1. Open `index.html` in a browser.
2. Or use a local live server extension in VS Code.

## Notes

- The movie data is mocked in `app.js`.
- You can replace the sample titles and colors with your own content.
